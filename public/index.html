Keyboard shortcuts

        
Ctrl-A: Select the whole content of the editor.

        
Ctrl-D: Deletes the whole line under the cursor, including newline at the end.

        
Ctrl-Z: Undo the last change.

        
Ctrl-Y: Redo the last undone change.

        
Ctrl-Home: When multiple selections are present, this deselects all but the primary selection.

        
Ctrl-End: When multiple selections are present, this deselects all but the primary selection.

        
Ctrl-]: When multiple selections are present, this deselects all but the primary selection.

        
Ctrl-[: When multiple selections are present, this deselects all but the primary selection.

        
F9: Run the code through Maude.

        
F2: Clear the text area.
        
        
Ctrl-/: Toggle this window.

        
Note: On Mac, replace Ctrl with Cmd
